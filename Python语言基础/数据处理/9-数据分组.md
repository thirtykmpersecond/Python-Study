# 数据分组
数据分组是指根据数据分析对象对特征，按照一定的数据指标，把数据分为不同的区间来进行研究，以揭示其内在的联系和规律性。简单来说，就是新增一列，将原来的数据按照其性质归入新的类别中。
命令格式：
```python
cut(series,bins,right=True,labels=NULL)
```
+ Series:表示需要分组的数据
+ bins:表示分组的依据数据
+ right:表示分组的时候右边是否闭合
+ labels:表示分组的自定义标签，可以不自定义
